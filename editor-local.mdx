# 🖱️ Editor local

Este guia explica como editar configurações locais e arquivos do Athilio_Spotify.

## Recomendações

- Não altere arquivos ofuscados sem autorização.
- Utilize o editor local apenas em arquivos de configuração.
- Faça backup antes de qualquer alteração.

<Warning>
  Modificações indevidas em arquivos internos podem invalidar sua licença.
</Warning>
