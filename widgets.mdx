# 🧩 Widgets

Os widgets do Athilio_Spotify permitem trazer informações importantes de forma visual.

## O que você pode usar

- Indicadores de licença
- Painéis de status
- Avisos de erro
- Informações do servidor

<Tip>
  Widgets são úteis para monitoramento rápido e comunicação com sua equipe.
</Tip>
