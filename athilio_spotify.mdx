## ⚙️ Framework

Todos os nossos resources são compatíveis com os principais frameworks utilizados no Brasil, como **Creative**, **Creative Network**, **vRP**, **vRPEX** e derivados.

## 🛠️ Instalação

1. Extraia o recurso `athilio_spotify` para a pasta `resources/`.
2. Abra o arquivo `athilio_spotify/license.json`.
3. Insira sua licença:

```json
{
  "license": "SUA_LICENSE_AQUI"
}
```

4. Adicione ao `server.cfg`:

```cfg
ensure athilio_spotify
```

5. Reinicie o servidor.

<Note>
  Use artefacts atualizados e mantenha o `oxmysql` sempre na versão correta.
</Note>

## 🧩 Erros e Soluções

### Problema: licença inválida

Verifique:

- se a licença está correta no `license.json`
- se o IP do servidor foi autorizado
- se o nome da pasta está correto
- se o servidor tem acesso à internet

### Problema: recurso não inicia

Verifique:

- `ensure athilio_spotify` no `server.cfg`
- ordem de carregamento das dependências
- cache do FiveM
- logs do console do servidor

### Problema: interface não abre

Verifique:

- pasta `html/` presente
- arquivos `index.html`, `style.css`, `script.js`
- erros no console `F8`

## 🔐 Licença

A licença do Athilio_Spotify valida o recurso durante a inicialização do servidor.

- Perda de licença por descumprimento não é reembolsável.
