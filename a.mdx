# Dashboard

O painel da **Athilio Software** permite gerenciar sua licença do AthilioFy em tempo real de forma simples, segura e centralizada.

Através do dashboard você poderá acompanhar:

- Status da licença
- Tempo restante
- Chave de autenticação
- Histórico de validações
- Informações do servidor
- Redefinição de IP
- Status da ativação

---

# Informações da Licença

Após acessar o painel, você visualizará as informações da sua licença ativa:

```txt
Status da Licença: Ativo
Dias Restantes: 29 / 30 dias
Plano Atual: mensal
Servidor Autorizado: 0.0.0.0
```

---

## Credenciais

A chave exibida no painel é utilizada para autenticação do recurso e deverá ser inserida no arquivo `athilio_spotify/license.json`.

```json
{
  "license": "SUA_LICENSE_AQUI"
}
```

<Warning>
  Nunca compartilhe sua licença publicamente. Sua chave é vinculada ao IP do seu servidor e de uso exclusivo.
</Warning>

---

# Guia de Instalação Rápida

<Steps>
  <Step title="Faça o download do recurso">
    Baixe o resource oficial do AthilioFy através do painel.
  </Step>
  <Step title="Configure o arquivo de licença">
    Crie ou edite o arquivo `athilio_spotify/license.json` com sua chave.

    ```json
    {
      "license": "SUA_LICENSE_AQUI"
    }
    ```
  </Step>
  <Step title="Inicie o recurso">
    Adicione no `server.cfg`:

    ```cfg
    ensure athilio_spotify
    ```
  </Step>
</Steps>

# Histórico de Validações

O painel disponibiliza um histórico completo das últimas autenticações realizadas pela sua licença.

```txt
18/05/26 21:57:04 | 0.0.0.0 | 30120 | Sucesso | OK
18/05/26 20:58:16 | 0.0.0.0 | 30120 | Sucesso | OK
17/05/26 23:03:20 | 0.0.0.0 | 30120 | Falhou  | Licença não encontrada
```

Cada validação registra:

- Data e horário
- IP do servidor
- Porta utilizada
- Status da autenticação
- Mensagem retornada

---

# Redefinição de IP

Caso o servidor seja migrado, reinstalado ou tenha alteração de IP, será necessário realizar uma redefinição no painel. Isso atualiza o novo endereço autorizado sem necessidade de gerar uma nova chave.

<Tip>
  Sempre confirme o IP correto do VPS antes de realizar alterações no painel.
</Tip>
