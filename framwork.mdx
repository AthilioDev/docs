# Framework

O **AthilioFy** foi desenvolvido utilizando arquitetura **Standalone**, garantindo compatibilidade com praticamente qualquer estrutura de servidor FiveM moderna.

O sistema foi projetado para funcionar de forma independente, sem dependências obrigatórias de frameworks específicos.

---

## Compatibilidade

O AthilioFy funciona normalmente em:

- vRP
- vRPex
- Creative
- QBCore
- ESX
- Bases próprias
- Bases híbridas
- Estruturas customizadas

---

## Arquitetura Standalone

Toda a estrutura do AthilioFy opera de maneira isolada do framework principal do servidor.

Isso significa que o recurso:

- Não exige framework específico
- Não depende de exports externos obrigatórios
- Não utiliza dependências desnecessárias
- Pode ser integrado facilmente em qualquer base
- Mantém alta compatibilidade entre diferentes ambientes

---

## Estrutura Independente

O AthilioFy possui:

- Sistema próprio de sincronização
- Eventos independentes
- Interface NUI própria
- Comunicação isolada
- Sistema interno de autenticação
- Estrutura otimizada para performance

Essa arquitetura reduz conflitos entre recursos e melhora significativamente a estabilidade operacional do servidor.
