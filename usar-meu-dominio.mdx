# 🌐 Usar meu domínio

Neste guia você aprende a configurar e apontar seu domínio próprio para o Athilio_Spotify.

## Passo a passo

1. Configure o DNS do seu domínio.
2. Aponte o registro `A` ou `CNAME` para o servidor onde o produto está hospedado.
3. Verifique a propagação do DNS.
4. Atualize o painel com o domínio configurado.

<Tip>
  Utilizar um domínio próprio ajuda a deixar a experiência mais profissional e segura.
</Tip>
